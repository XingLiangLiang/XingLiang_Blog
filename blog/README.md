# :black_nib: 小桥酒馆

> 记录技术的新博客，采用Vue全家桶开发

## 特点
- Vue 2 / Vue-cli / axios / Less
- GitHub API 进行数据交互，无需后端服务器
- 无需本地打包，`git push`直接部署 (使用TravisCI)
- 在GitHub页面上托管
